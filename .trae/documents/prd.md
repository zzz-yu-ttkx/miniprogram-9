## 1. Product Overview
电子名片是一款个人/企业展示型网页应用，帮助用户通过精美、专业的数字名片展示个人信息、公司介绍和案例作品，提升个人品牌形象。

## 2. Core Features

### 2.1 User Roles
| Role | Registration Method | Core Permissions |
|------|---------------------|------------------|
| Viewer | None | View all content |

### 2.2 Feature Module
1. **电子名片主页**: 个人信息展示、联系方式、公司介绍、作品案例展示

### 2.3 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 电子名片主页 | 头像区域 | 展示个人头像，支持圆形裁剪展示 |
| 电子名片主页 | 联系方式 | 展示电话、邮箱、微信等联系信息，支持一键拨打电话 |
| 电子名片主页 | 公司介绍 | 展示公司名称、职位、公司简介 |
| 电子名片主页 | 图片展示 | 展示个人或公司相关图片 |
| 电子名片主页 | 案例展示 | 展示过往项目案例，支持点击查看详情 |

## 3. Core Process
用户打开电子名片页面 → 浏览个人信息和公司介绍 → 查看案例作品 → 通过联系方式进行沟通

## 4. User Interface Design
### 4.1 Design Style
- 主色调：深蓝色系 (#1e3a5f)，搭配金色点缀 (#d4af37)
- 按钮风格：圆角矩形，渐变背景
- 字体：使用 Google Fonts 的 'Noto Sans SC' 中文无衬线字体
- 布局风格：卡片式布局，居中对齐
- 图标风格：简洁现代的线性图标

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 电子名片主页 | 头像区域 | 圆形头像，悬浮阴影效果 |
| 电子名片主页 | 联系方式 | 图标+文字，悬停高亮 |
| 电子名片主页 | 公司介绍 | 标题+段落，卡片背景 |
| 电子名片主页 | 图片展示 | 瀑布流/网格布局，图片懒加载 |
| 电子名片主页 | 案例展示 | 卡片式展示，悬停放大效果 |

### 4.3 Responsiveness
移动端自适应：单列布局，点击弹出详情模态框
桌面端：多列网格布局，悬停预览效果

不适用
